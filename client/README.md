# RefineryIQ Frontend

Vite + React + TypeScript + Tailwind CSS frontend for the RefineryIQ platform.

## Development

```sh
npm install
npm run dev
```

## Build

```sh
npm run build
npm run preview
```

## Notes

- Backend API base URL is configured via `VITE_API_BASE`.
- The UI expects the backend APIs to be running for live data.
